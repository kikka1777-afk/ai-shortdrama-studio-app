Writer→Director→Storyboard→Stylist 串联工作流。在线生成经同源 /api/generate 代理调用 OpenAI 兼容接口或 Azure； API Key 由管理员在 Vercel 环境变量里统一配置。

设置模型接口

支持 OpenAI 兼容接口与 Azure OpenAI。配置只存在你本地浏览器 localStorage。留空则只能跑内置示例（除非已在 Vercel 设了环境变量）。

提示：请求由同源的 /api/generate 代理转发，已避开跨域(CORS)问题。你也可以不在这里填，改在 Vercel 项目设置 → Environment Variables 里设 OPENAI_API_KEY（和可选 OPENAI_BASE_URL），同事打开就不用各自填 key。
